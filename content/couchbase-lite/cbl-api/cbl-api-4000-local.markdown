# Local Document Resources

Local document resources enable you to create documents that are not replicated to other databases. You can use local documents to store configuration or other information  The following table lists the local document resources:

|HTTP method | URI pattern | Description  |
| ------	| ------	| ------	|  
| `POST`    | `/{db}/local/{local-doc}` | Creates a new document in the database  
| `GET`| `/{db}/{doc}` | Retrieves a specific document |
