Couchnode overview

The Couchbase Node.js SDK, known as Couchnode, enables you to interact with Couchbase Server from a Node.js application.

Couchnode consists of a native JavaScript library and a C/C++ add-on that uses the Couchbase C SDK (libcouchbase) internally. The Couchnode SDK includes the following components:

  • Couchnode client
  • Developer guide (you're reading it now!)
  • Couchnode API reference
  • Sample applications (beer database, CouchQA)

Contributing

Couchbase welcomes community contributions to Couchnode. The Couchnode source code is available on GitHub. If you would like to contribute code, read the contributor's guide for details.
