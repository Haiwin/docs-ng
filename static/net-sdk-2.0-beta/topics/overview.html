.NET SDK 2.0 beta

The Couchbase .NET SDK 2.0 is the official .NET client library for Couchbase Server. Written entirely in the C# programming language, it targets .NET Framework 4.5 and later and Couchbase Server 2.5 and later.

This is the documentation for the SDK and it groups features into sections. Within each feature section, you will find concept or task references to additional information or topics, and other topics that are identified as relating to the feature in discussion.

Until the SDK reaches general availability (GA), expect the content and structure of the documentation to change. Being a work in progress, we greatly appreciate your feedback. Please send your comments, requests, and suggestions to jeffry.morris@couchbase.com.
