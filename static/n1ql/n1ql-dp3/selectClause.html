SELECT clause

The SELECT clause queries data from the Couchbase database.

Syntax

SELECT 
   ALL result-expr
   DISTINCT result-expr
   RAW expr
   result-expr

Compatibility

Compatible with Couchbase Server 2.2

Example

The following example is based on the N1QL tutorial.

SELECT 'Hello World' AS Greeting

Returns:

{
  "resultset": [
    {
      "Greeting": "Hello World"
    }
  ]
}
