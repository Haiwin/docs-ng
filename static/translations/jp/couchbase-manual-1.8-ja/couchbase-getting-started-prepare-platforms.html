2.1.1. 対応プラットフォーム

次のオペレーティングシステムがサポートされています。

  • RedHat Enterprise Linux 5.2 (非推奨)追加のサードパーティライブラリとパッケージが必要です

  • RedHat Enterprise Linux 5.4 (32-bit および 64-bit)

  • Ubuntu Linux 10.04 (32-bit および 64-bit)

  • Ubuntu Linux 11.10 (32-bit および 64-bit) 開発者のみ

  • Windows Server 2008 (32-bit および 64-bit)

  • Mac OS X 10.5 以上 (最低でも)、 10.6 以上を推奨 (64-bit のみ) 開発者のみ
