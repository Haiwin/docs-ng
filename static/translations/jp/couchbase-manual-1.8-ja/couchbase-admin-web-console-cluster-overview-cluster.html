6.1.1. Cluster

Clusterセクションは、クラスタのRAMとディスクの使用状況に関する情報を提供します。

RAMの情報に関しては、下記を含めて、RAMの状況がグラフィカルに表示されます。

  • Total in Cluster

    クラスタで構成されているRAMの合計です。これは、クラスタ内のすべてのサーバに構成されているメモリの総量です。

  • Total Allocated

    クラスタ内のデータバケットに割り当てられたRAM量。

  • Unallocated

    クラスタ内のデータバケットに割り当てられていないRAM量。

  • In Use

    実際に使用中のすべてのバケットのメモリ量（つまり、データが実際に格納されている）。

  • Unused

    （使用可能ではあるが、）データ格納に使用されていないメモリ量。

Disk Overviewセクションは、クラスタ全体のディスクストレージ容量と同等な要約情報を提供します。

  • Total Cluster Storage

    クラスタ全体で、データの格納に使用可能なディスクストレージの総量。

  • Usable Free Space

    ディスク上に情報を格納するための使用可能な領域。非Couchbaseファイルの格納も考慮に入れて、設定されたパス上で使用可能な領域を示しています。

  • Other Data

    Couchbase情報以外のデータによって使用されているディスク領域。

  • In Use

    実際にディスク上に情報を格納して使用されているディスク領域。

  • Free

    ディスク上にオブジェクトを格納するため使用可能な空き領域。
