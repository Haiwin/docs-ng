6.2. Webコンソールの統計情報

Data Bucketのモニタ表示内で、全Couchbase Serverクラスタの情報がデフォルトで表示されます。その情報は、選択されたバケットについて、クラスタ内のすべてのサーバノードから集計されます。

標準のData Bucketsのモニタ画面を以下に示します。個々のグラフは、選択した時間間隔にしたがって、リアルタイムで更新されます。

次の機能がこの画面で使用可能です。また、これらの機能は、Webコンソール内のすべてのグラフおよび統計情報の表示で共通です。

  • Bucket Selection

    データバケットの選択リストで、クラスタ上に設定したバケットの中から、どのバケットをグラフ表示の対象とするかを選択することができます。

  • Server Selection

    サーバの選択オプションは、クラスタ全体に対してではなく、個々のサーバの表示を有効にすることができます。「サーバーノード」に表示される個々のノードを選択できます。 All Server Nodesを選択すると「データバケット」ページが表示されます。

  • Interval Selection

    メイングラフの最上部にあるInterval Selectionは、ページに表示されるすべてのグラフの間隔表示を変更します。例えば、Minuteを選択すると、最後の1分の情報が、継続的に更新し表示されます。

    注記

    選択された間隔が増加するにつれて、表示される統計データの量は、クラスタが稼働している時間の長さに依存します。

  • 統計グラフの選択

    ディスプレイ内のすべてのグラフは、同時に更新されます。小さなグラフのいずれかをクリックすると、そのグラフが、ページのメイングラフとして表示されるようになります。

  • Individual Server Selection

    小さな統計グラフのいずれかの隣にある青い三角形をクリックすると、特定のクラスタ全体の統計情報を表示する初期表示から、個々のサーバの統計情報表示へ変更できます。
