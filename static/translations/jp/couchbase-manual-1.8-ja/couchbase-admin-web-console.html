CouchbaseのWebコンソールは、Couchbaseを管理するための主要な手段です。Webコンソールは、次の機能を提供します。

  • Cluster Overviewは、Couchbaseクラスタの状態を素早く把握することができます。

    詳細については、「クラスタの概要」 を参照してください。

  • Data Bucketsは 、新しいバケットの作成、既存の設定の編集を含むデータバケット構成へアクセスできます。またバケットの状態に関する詳細な分析や統計を提供します。

    こちらも参照してください。「データバケット」

  • Server Nodesは、アクティブなノードとその構成、および状態に関する詳細情報を提供します。ノードをフェールオーバーしたり、クラスタからノードを削除したり、サーバー固有のパフォーマンスと監視統計を表示することもできます。

    こちらも参照してください。「サーバーノード」

  • Logは、エラーや問題を表示することができます。

    詳細はこちらを参照してください。 「ログ」

  • Settingsでは、コンソールとクラスタの設定を変更できます。

    詳細はこちらを参照してください。 「設定」

CouchbaseのWebコンソールのナビゲートセクションに加えて、Webコンソール内に、次のような機能があります。

  • 更新通知

    インストールされているCouchbase Serverで有効な更新があるときは、更新通知が警告して知らせます。この機能の詳細については、次を参照してください。 「アップデート通知」

  • 警告と警報

    Webコンソールを介して、クラスタ内でどこに対処するべき問題があるかを通知します。警告とアラートは、こちらで設定することができます。 「設定」

    警告と警報の詳細については、次を参照してください。 「警告とアラート」
