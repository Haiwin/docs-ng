6.6.2. 自動フェイルオーバーの設定

Auto-Failoverの設定では、自動フェイルオーバーの有効化と、クラスタノードの障害が検出されてから、自動フェイルオーバープロセスが開始されるまでのタイムアウトを設定します。

自動フェイルオーバーを有効にするには、 Enable auto-failoverチェックボックスをチェックしてください。自動フェイルオーバーが開始されるまでの時間を設定するには、秒数をTimeoutボックスに入力してください。デフォルトのタイムアウトは30秒です。

図6.16 Webコンソールの設定 - 自動フェイルオーバー


自動フェイルオーバーの詳細については、「自動フェイルオーバー」を参照してください。
