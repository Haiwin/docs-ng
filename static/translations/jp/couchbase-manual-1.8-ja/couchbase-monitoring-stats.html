12.4. Couchbase Server 統計情報

12.4.1. RESTインターフェイスによる統計情報
12.4.2. Couchbase Server ノードの統計情報

Couchbase Serverは、複数のレベルでクラスタ全体の統計を提供します。これらは、定常的なモニタリング、キャパシティプランニングや、パフォーマンス特定を識別するために使用できます。Web UIが最も視覚的な統計情報ですが、RESTインタフェース、プロキシや個々のノードは直接アクセス可能な統計インタフェースがありま。
