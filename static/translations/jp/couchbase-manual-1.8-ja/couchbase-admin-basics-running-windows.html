3.2.2. Windowsでの起動とシャットダウン

Windowsでは、Couchbase ServerはWindowsサービスとしてインストールされています。Couchbase Serverを起動、停止するには、Windowsのタスクマネージャ内の[サービス]タブを使用します。

注記

また、Power UserかAdministrator権限が必要となります。もしくは別々にCouchbase Serverを起動、停止するサービスを管理する権限が与えられている必要があります。

初期設定では、マシンの起動時にサービスが自動的に起動します。手動でサービスを起動するにはWindowsタスクマネージャを開き、[サービス]タブを選択するか、[スタート]メニューから、[実行]を選択し、Services.mscを入力してサービス管理コンソールを開きます。

サービス管理コンソールを開いたら、必要に応じて右クリックでサービスを起動または停止を選択し、CouchbaseServerサービスを見ます。サービスがブート中に自動的に起動されないように設定を変更することができます。

また、システムのnetコマンドを使用すし、コマンドラインからサ ​​ービスを起動および停止することができます。例として、 Couchbase Serverを起動： 

shell> ​​net start CouchbaseServer

Couchbase Serverを停止： 

shell> net stop CouchbaseServer

起動および停止スクリプトはCouchbase Serverインストールディレクトリのbinディレクトリにあります。このスクリプトを使用してサーバを起動するには： 

shell> C:\Program Files\Couchbase\Server\bin\service_start.bat

付属のスクリプトを使用してサーバーを停止するには： 

shell> C:\Program Files\Couchbase\Server\bin\service_stop.bat
