Couchbase Serverには、Couchbase Serverクラスタまたはサーバを管理、監視するために使用できるいくつかのコマンドラインツールがあります。 すべての操作は、（使用可能なところでは）適切な要求にマッピングされます。8章Couchbase 管理 REST API

さまざまな機能と操作を実行するコマンドラインツールがいくつかあります。以下のセクションで個別に説明しています。ツールは用途別のディレクトリに配置されています。

表7.1 Couchbase Serverのコマンドラインツール

ツール内容
couchbase-cli Couchbase Serverと通信し管理するためのメインツール。
cbstats Couchbase Serverノードのローカルな統計情報を計測するためのツール。
cbflushctl ディスクへの永続化動作を制御するためのツール。
tap.py Couchbase ServerノードのTAPインタフェースにアクセスするためのツール。
cbvbucketctl Couchbase ServerノードのvBucket状態を制御するためのツール。
cbdbmaint Couchbase ServerノードのSQLiteデータベースを管理するためのツール。
cbcollect_info Couchbase Serverノードからの多くの統計情報を収集するための支援ツール。
