6.6.1. アップデート通知の設定

Update Notification画面内で、Enable software update notificationsのチェックボックスをチェックすることにより、アップデート通知を有効または無効にすることができます。このオプションを変更したら、変更を保存するために、Saveをクリックしてください。

アップデート通知が無効になっている場合、アップデート通知画面は、現在インストールされているバージョンを表示し、アラートは出ません。

図6.15 Webコンソールの設定 - アップデート通知


アップデート通知機能の詳細については、「アップデート通知」を参照してください。
