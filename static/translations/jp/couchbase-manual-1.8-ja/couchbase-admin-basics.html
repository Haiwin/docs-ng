この章は、Couchbase Severクラスタの管理について全てを網羅しています。クラスタ管理は、主に次の3つの方法により行います:

  • Couchbase Webコンソール

    Couchbaseには、クラスタの管理と統計情報へのアクセスを提供する組み込みWebサーバと、管理インターフェイスが含まれています。

    詳細はこちらをご覧下さい。6章Couchbase管理Webコンソール

  • コマンドラインツールキット

    Couchbaseクラスタとの通信および制御を可能とするコマンドラインツールの数々はCouchbaseのパッケージとして提供されています。

    詳細はこちらをご覧下さい。7章Couchbaseコマンドラインインタフェース

  • Couchbase REST API

    Couchbase ServerはCouchbaseのクラスタの管理と監視をするために、HTTPで通信することができるツールを利用可能なRESTful APIが含まれています。

    詳細はこちらをご覧下さい。8章Couchbase 管理 REST API
