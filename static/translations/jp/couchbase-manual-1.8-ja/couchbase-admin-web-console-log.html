6.7. ログ

6.7.1. 診断レポート

Web UIのLogセクションでは、Couchbaseクラスタ内の動作とエラーを識別できるように、Couchbaseサーバの組み込みのイベントログを表示することができます。

図6.18 Webコンソールログビューア


CouchbaseWebコンソールには、各ログに記録されたイベントに関する以下の情報が表示されます。

名前内容
イベントアラートを発生させたイベント。
モジュールイベントが発生したソフトウェアモジュールコー​​ドとイベントに関連する内部アラートコード。
サーバーノードログエントリを発生させたサーバノード。
時刻イベントが発生した時刻と日付。
