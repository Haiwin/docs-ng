概要

これはCouchbase Server 1.8と互換性のある、Couchbase Javaクライアントライブラリのバージョン 1.0 のマニュアルです。

このマニュアルはCouchbase Java クライアントライブラリ（couchbase-clientと、 spymemcached）を利用する際の主要な機能へのリファレンスと、ベストプラクティスを提供します。内容は主要なAPIのリファレンスであり、全てのAPI機能に関する完璧な解説ではありません。

表1 Product Compatibility for Couchbase SDK Java

ProductCompatibleAll Features
Couchbase Server 1.8✓✓
Couchbase Server 2.0✓✗
