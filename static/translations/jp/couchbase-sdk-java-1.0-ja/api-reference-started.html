3.1. Couchbaseバケットへの接続
3.2. ホスト名とポートを利用したSASL接続
3.3. 接続をシャットダウンする

クライアントライブラリは Couchbase と Memcached クライアント両方への一貫したインタフェースを提供します。JavaアプリケーションとCouchbase あるいは Memcached サーバ間のインタフェースは一つのオブジェクト、CouchbaseClient クラスのインスタンスを導入することで提供されます。

このクラスの新しいオブジェクトを生成すると設定された各サーバへの接続をオープンし、値の設定、取得や更新に関する全てのサーバとのコミュニケーションを処理します。接続先アドレスを指定し、CouchbaseClientオブジェクトを生成するために、様々なメソッドが用意されています。
