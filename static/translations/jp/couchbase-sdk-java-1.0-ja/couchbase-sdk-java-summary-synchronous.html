4.1. 同期メソッド呼び出し

JavaクライアントライブラリはCouchbaseサーバに対して直接APIを呼び出すことで、主要なCouchbase API メソッドをサポートしています。これらの直接的なメソッドはインスタンスストレージとして、Couchbase key/value ペアの取得および更新が可能です。

例えば、次のコードは単一の key/value ペアを保存し、取得します： 

client.set("someKey", 3600, someObject);

Object myObject = client.get("someKey");

上記のコードでは、クライアントの get() 呼び出しは、適切に構成された Couchbase サーバからの応答を受け取るまで待って、値または例外を返します。
