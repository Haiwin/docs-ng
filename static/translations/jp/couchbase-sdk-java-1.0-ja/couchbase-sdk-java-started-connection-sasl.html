3.2. ホスト名とポートを利用したSASL接続

SASL を利用してCouchbaseに対するセキュアな接続を提供したいのであれば、SASL接続種別、ユーザバケットとパスワードを定義可能な CouchbaseConnectionFactory が利用できます。

Couchbase への接続はSASLのプロトコルをベースとして行われます。次の例は前述の例と同様ですが、CouchbaseConnectionFactoryの利用が異なります。 

 
        List<URI> baseURIs = new ArrayList<URI>();
        baseURIs.add(base);
        CouchbaseConnectionFactory cf = new
                CouchbaseConnectionFactory(baseURIs,
                    "userbucket", "password");

        client = new CouchbaseClient((CouchbaseConnectionFactory) cf);
