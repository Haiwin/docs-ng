1.1. Couchbaseクライアントライブラリのダウンロード

クライアントライブラリとその依存関係をダウンロードして、それらを正しくクラスパス上に配置してください。クライアントJARファイルと依存関係を実行するためのインストール方法はこちらを参照してください。Couchbase用Javaクライアントライブラリから直接ダウンロードできます。

これらのファイルはJavaアーカイブ（.jar）ファイルで、Java実行環境で利用できます。
