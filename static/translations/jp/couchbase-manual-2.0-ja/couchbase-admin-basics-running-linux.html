3.2.1. Linuxでの起動とシャットダウン

Linux上でCouchbase Serverは、スタンドアロンアプリケーションとしてインストールされ、標準の制御スクリプト（/etc/init.d/couchbase-server）を使用した起動時にバックグラウンド(デーモン)プロセスとして実行することをサポートします。起動スクリプトはいずれのLinuxディストリビューション（Debian / Ubuntu、RedHat / CentOS）のパッケージにも含まれており、自動的にインストールされます。デフォルトでのCouchbase Serverは、ランレベル2、3、4、5で自動的に起動するように構成されており、ランレベル0、1、6ではシャットダウンされます。

手動で起動するには、以下のスクリプトを実行します：

shell> sudo /etc/init.d/couchbase-server start

手動で終了するには、以下のスクリプトを実行します：

shell> sudo /etc/init.d/couchbase-server stop
