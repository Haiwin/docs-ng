概要

このマニュアルは、Couchbase Server _ meta_version__系のインストール、監視、管理インターフェイスおよび関連ツールについて記載しています。

対応するMoxi製品については、Moxi 1.8系を使用してください。Moxi 1.8 マニュアルを参照してください。
