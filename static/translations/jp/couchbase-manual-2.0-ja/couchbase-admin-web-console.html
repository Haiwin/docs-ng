CouchbaseのWebコンソールは、Couchbaseを管理するための主要な手段です。Webコンソールは、次の機能を提供します。

  • クラスタの概要：Couchbaseクラスタの状態を素早く把握することができます。

    詳細は「 クラスタの要約の表示 」をご覧ください。

  • データバケット：データバケットの設定の表示および更新ができます。新しいバケットを作成したり、既存の設定を編集し、バケットに関する詳細な統計情報を見ることができます。

    「 データバケットの表示 」も参照してください。

  • サーバー・ノード：アクティブノードと、それらの設定およびアクティビティを参照することができます。このタブでは、ノードをフェールオーバすることや、クラスタから削除することもでき、また、サーバ固有のパフォーマンスの表示や、クラスタの統計情報を監視することもできます。

    「 サーバノードの表示 」も読んでください。

  • 再生回数：インデックスとデータ問い合わせのためのビュー関数を作成したり、管理することができます。ここでは、ビューからの結果をプレビューすることもできます。

    Webコンソールでビューを編集するには、「ビューエディタの利用」を参照してください。一般的なビューの詳細については、9章ビューとインデックスを参照してください。

  • ドキュメント：このタブではドキュメントの作成や編集ができます。これにより、データ・バケットに格納されているドキュメントを参照および変更することができ、ビューを動作させるときに役に立つ可能性があります。

    「ドキュメントエディタの利用」を参照してください。

  • ログ：エラーや問題を表示します。

    詳細情報については、「 ログ 」を参照してください。

  • 設定：このタブではコンソールおよびクラスタ設定を行うことができます。

    詳細情報については、「 設定 」を参照してください。

CouchbaseのWebコンソールのナビゲートセクションに加えて、Webコンソール内に、次のような機能があります。

  • 更新通知

    更新通知は、インストールされているCouchbase Serverで利用可能な更新があることを示しています。この機能の詳細については、「更新通知」を参照してください。

  • 警告とアラート

    クラスタ内で対処すべき問題があるときにWebコンソールでの警告とアラートによって気づくでしょう。警告とアラートは、「 設定 」経由で設定可能です。

    警告とアラートの詳細については、「警告とアラート」を参照してください。
