6.1.2. バケットの表示

バケットセクションは、１秒あたりの操作数と１秒あたりのディスクフェッチ数という２つのグラフが表示されます。

図6.3 Webコンソール — クラスタ概要 — バケット


１秒あたりの操作数はそのデータストアにオブジェクトが格納されたもしくは、取り出されたという観点からクラスタの活動レベルについて情報を提供します。

１秒あたりのディスクフェッチ数はRAMに格納された情報を使用する代わりに情報を取り出すためにCouchbaseがどのくらい頻繁にディスクへアクセスしているかを示します。
