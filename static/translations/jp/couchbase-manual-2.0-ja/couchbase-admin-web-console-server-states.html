6.2.1. サーバ状態の理解

Couchbase Serverノードは現在の挙動や状態に応じていくつかの状態に分類できます。表示される状態は、次のとおりです。

  • 稼働中

    ホストは起動していて、ノード間でデータを複製しクライアントからのリクエストに応答しています。

  • 停止中

    ホストはダウンしていて、ノード間でデータを複製せず、クライアントからのリクエストにも応答しません。

    図6.9 Webコンソール — 停止ステータス


  • 保留中

    ホストは起動していますが、データをRAMにロードしている最中で、クライアントからのリクエストには応答しません。データをRAMにロードし終えたら、クライアントアクセスが可能になります。

    図6.10 Webコンソール — 保留ステータス


Webコンソール内でManage: Server　NodesとMonitor: Server Nodesスクリーンの両方を使用して現在のサーバステータスを監視することができます。
