9.10.5. SQL LIMIT と OFFSETの翻訳

SQLでは、クエリのLIMITとOFFSET句はページングメカニズムとして使用されます。たとえば、次のように使用します：

SELECT recipeid,title FROM recipes LIMIT 100

データベースから最初の100行を取得するためには上記のように、そしてレコードのそのあとのグループを取得するためにはOFFSETを使用してください： 

SELECT recipeid,title FROM recipes LIMIT 100 OFFSET 100

Couchbase Serverで、limitとskipパラメータをクエリで指定するとき、同じような基本機能を提供します：

?limit=100&skip=100

skipに高い値を指定すると、性能に悪影響を与えることがあります。ページングの仕組みの他の例については、「ページネーション」を参照してください。
