9.11.1. 幾何学データの追加

GeoCouchはGeoJSON仕様を利用した幾何学情報の格納をサポートしています。ポイントデータのストレージ形式は、ビューインデックスの生成中にサポートされる幾何学タイプと一緒で任意です。

たとえば、単純な位置データを格納するための2次元ジオメトリを使用することができます。任意のフィールド名を使用して、Couchbaseのドキュメントにこれらを追加することができます。慣習では、ポイント位置の2次配列の単一フィールドを使用しますが、2つの独立したフィールドや地理空間インデックスへの情報を集めたビューのような複雑な構造を使用することもできます。

たとえば、都市の位置情報をバケットに投入するために、バケットに送信されるドキュメントは、以下のような形式にすることができます：

JSON
{
"loc" : [-122.270833, 37.804444],
"title" : "Oakland"
}
