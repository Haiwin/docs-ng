8.6.7. バケットパラメータの変更

バケットのURIに更新するパラメータをPOSTすることで、バケット作成時に利用したパラメータを更新することができます。多くの場合、設定しないのと同じ意味になるので、リクエスト内でパラメータを省略しないでください。現在のバケットの設定を取得するためのリクエストして、必要な変更を行って、それからバケットURIへPOSTリクエストをすることを推奨します。

たとえば、customerバケットを編集するには：

shell> curl -v -X POST -u Administrator:Password -d name=customer \
    -d flushEnabled=0 -d replicaNumber=1 -d authType=none \
    -d ramQuotaMB=200 -d proxyPort=11212 \
     http://localhost:8091/pools/default/buckets/customer

バケットを作成する際に使用可能なパラメータは同じように使用可能です。bucket parametersを参照してください。

リクエストが成功した場合、HTTPレスポンス 200が空のデータ内容で返されます。

警告

REST APIを介してバケットの名前を変更することはできません。
