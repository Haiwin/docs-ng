Couchbase Serverのインスタンス、つまりノードは、Couchbase Serverを実行する物理マシンまたは仮想マシンです。各ノードはクラスタのメンバとして存在します。

Couchbaseのクラスター存在するノードに関する情報を表示するには、このリクエストを使用します。

shell> curl -u admin:password 10.4.2.4:8091/pools/nodes
