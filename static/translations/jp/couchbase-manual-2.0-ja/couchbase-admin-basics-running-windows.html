3.2.2. Windowsでの起動とシャットダウン

Windowsでは、Couchbase ServerはWindowsサービスとしてインストールされます。Couchbase Serverを起動、停止するには、Windowsのタスクマネージャ内のサービスタブを使用します。

注記

また、Power UserかAdministrator権限が必要となります。もしくは別々にCouchbase Serverを起動、停止するサービスを管理する権限が与えられている必要があります。

初期設定では、マシンの起動時にサービスが自動的に起動します。手動でサービスを開始するには、Windowsタスクマネージャを開き、サービスタブ選択するか開始を選択し 、実行を選択します。サービスの管理コンソールを開くにはServices.mscを入力します。

サービス管理コンソールを開いたら、CouchbaseServerサービスを探して必要に応じて右クリックでサービスを起動または停止を選択します。サービスがブート中に自動的に起動されないように設定を変更することもできます。

また、システムのnetコマンドを使用し、コマンドラインからサービスを起動および停止することができます。以下に例を示します。 Couchbase Serverを起動：

shell> net start CouchbaseServer

Couchbase Serverを停止：

shell> net stop CouchbaseServer

起動および停止スクリプトはCouchbase Serverインストールディレクトリのbinディレクトリにあります。このスクリプトを使用してサーバを起動するには：

shell> C:\Program Files\Couchbase\Server\bin\service_start.bat

付属のスクリプトを使用してサーバを停止するには：

shell> C:\Program Files\Couchbase\Server\bin\service_stop.bat
