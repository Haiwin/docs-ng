4.4.1. 監視が必要なUIの統計

次のグラフをCouchbaseコンソールの監視対象に追加できます。これらのグラフは、CouchbaseのWebコンソール上でBucket Detailsの上部にあるConfigure Viewリンクをクリックして、選択することができます。

  • Disk write queues

    増加していないこと（実際の値はアプリケーションとデプロイ環境に依存します）

  • Ram ejections

    突然のスパイクがあってはなりません。

  • Vbucket errors

    vBucketのエラーが増加していないこと

  • OOM errors per sec

    これは0でなければなりません。

  • Temp OOM errors per sec

    これは0でなければなりません。

  • Connections count

    長期にわたる運用で大きく変わらないこと

  • Get hits per second

  • Get misses per second

    Get hits per secondよりも少ないこと
