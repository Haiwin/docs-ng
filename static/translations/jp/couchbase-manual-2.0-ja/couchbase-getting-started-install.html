2.2. Couchbase Serverのインストール

2.2.1. Red Hat Linuxへのインストール
2.2.2. Ubuntu Linuxへのインストール
2.2.3. Microsoft Windowsへのインストール
2.2.4. Mac OS X インストール

お使いのマシンにCouchbase Serverをインストールするにはhttp://www.couchbase.com/downloadsから、選択したプラットフォームに適したパッケージをダウンロードする必要があります。各プラットフォームに対応するプラットフォーム固有の指示に従ってください。

注記

既にCouchbase Serverがインストールされているが、アップグレードインストールを実行したくない場合、インストールを開始する前に、Couchbase Server及びすべての関連データを削除する必要があります。Couchbase Serverをアンインストールする方法の詳細については、付録B Couchbase Serverのアンインストールを参照してください 。

既存のデータセットを保持したままアップグレードインストールを実行するには、「Couchbase Server 2.0 へのアップグレード」 を参照してください。
