10.6. Couchbase Server Moxiの統計情報

Moxiは、memcachedプロトコルのサポートの一環として、memcachedの statsコマンドをサポートします。　通常のmemcachedのクライアントはmemcachedの statsコマンドを使用して統計をリクエストできます。statsコマンドはオプションの引数を受け取り、Moxiの場合には統計プロキシサブコマンドがあります。Moxiで表示可能な統計情報の詳細な説明がここに記載されています。

例えば、あるシンプルなクライアントは一般的に利用可能なnetcatを使います（出力は省略記号で省略しています）。

$ echo "stats proxy" | nc localhost 11211
STAT basic:version 1.6.0
STAT basic:nthreads 5
…
STAT proxy_main:conf_type dynamic
STAT proxy_main:behavior:cycle 0
STAT proxy_main:behavior:downstream_max 4
STAT proxy_main:behavior:downstream_conn_max 0
STAT proxy_main:behavior:downstream_weight 0
…
