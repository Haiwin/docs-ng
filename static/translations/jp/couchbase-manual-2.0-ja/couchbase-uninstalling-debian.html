B.2. Debian/Ubuntu Linuxシステムでのアンインストールについて

Ubuntu Linuxでアンインストールをするには、次のコマンドを実行します。

shell> sudo dpkg -r couchbase-server

dpkgを使用したパッケージアンインストールの詳細については、Ubuntuのマニュアルを参照してください。

また、必要に応じてデータファイルを削除する必要があります。デフォルトのインストールパスは/optです。データファイルの格納場所を指定した場合は、別途、システムからそれぞれのデータディレクトリを削除する必要があります。
