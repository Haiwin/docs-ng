    Installing as non-root, non-sudo

    Installing on Mac OS X as non-root, non-sudo is for development purposes only.

    There may be cases when you want to install the server as a non-root, non-sudo user. If you perform a non-sudo, non-root installation, you can run Couchbase Server and all Couchbase command-line tools.

    To install as non-root, non-sudo:

    1. After downloading Couchbase Server, open Terminal and go to the Downloads directory:

      cd ~/Downloads/

    2. Unzip the package containing Couchbase Server:

      open couchbase-server-enterprise_x86_64_2.1.0.zip

    3. Move Couchbase App to your /Applications folder:

      mv couchbase-server-enterprise_x86_64_2.1.0/Couchbase\ Server.app /Applications/

    4. Start the server from Terminal:

      open /Applications/Couchbase\ Server.app

      This enables you to use Couchbase Server as a non-sudo, non-root user.

    5. To stop the server, click on the Couchbase icon in the menu bar and select Quit Couchbase Server.
