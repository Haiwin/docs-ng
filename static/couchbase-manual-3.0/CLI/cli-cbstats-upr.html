    DCP statistics (CLI)

    Statistics for the Database Change Protocol (DCP) are obtained via the CLI with the cbstatstool.

    Table 1. DCP statistics
    Option Description
    upr tbd
    upr -takeover vb name tbd
    upragg tbd

    Syntax

    cbstats HOST:PORT upr

cbstats HOST:PORT upr -takeover vb name

cbstats HOST:PORT upragg
