    Couchbase 3.0 beta introduction

    This documentation set is for the Couchbase Server 3.0 Beta.

    The documentation groups the new features in one section. Each feature section contains concepts or tasks, references to additional topics, and identifies topics that are impacted one way or another by the features. In some cases, existing topics may be modified or replaced.

    Until General Availability (GA), the documentation may and, most likely, will undergo significant updates and restructuring.

    PLEASE feel free to submit comments, requests, and suggestions to the Documentation Team at docs@couchbase.com.
